# @aw-webflow/masterclass_gated_page

## Usage via jsDelivr CDN

```html
<script src="https://cdn.jsdelivr.net/npm/@aw-webflow/masterclass_gated_page@1.0.0/script.min.js"></script>
```

## Deployment Workflow

1. Push changes to GitHub
2. Bump the version in `package.json`
3. Run `npm publish --access public`
4. Update the version number in the Webflow script tag

## Local Development

```bash
npm install
npm start
```
